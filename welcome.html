Talko

Please grant access to your microphone in order to use this extension!

Start Talko by refreshing the page and saying "Talko, start." For a full list of commands, scroll down or say "help" to bring you back to this command list.

Commands

  • "Click ____" :Clicks link most similar to the one specified
  • "Search _____": Opens Bing search for the specified search terms
  • "Go back": Navigates to previous web page
  • "Scroll up/down": Scrolls up or down the web page
  • "New tab": Opens a new blank tab
  • "Print": Opens a popup to print the webpage
  • "Full Screen": Makes the webpage full screen
  • "Talko, Stop": Disables Talko until "Talko, Start" is said again
  • "Help": Brings you back to this page
